Landing Pages Collection
Este repositorio contiene una colección de diversas páginas de aterrizaje (landing pages) desarrolladas de forma modular. Cada proyecto está organizado en su propia carpeta con su estructura independiente de estilos, imágenes y maquetación.

📂 Estructura del Proyecto
Cada carpeta de "landing" sigue una estructura estándar:

/css: Hojas de estilo específicas del diseño.

/imagenes o /media: Recursos visuales y assets.

index.html: Archivo principal de la página.

🚀 Proyectos Incluidos
La colección incluye las siguientes versiones:

Landing 1: Estructura básica con master.css.

Landing 2 & 3: Diseños personalizados con assets específicos.

Landing 4: Versión con estilos específicos en landing4.css.

Landings 14, 33, 40, 45: Variaciones adicionales del catálogo.

🛠️ Tecnologías utilizadas
HTML5: Marcado semántico.

CSS3: Estilos y layouts.

Instrucciones de uso:
Simplemente abre el archivo .html de cualquier carpeta en tu navegador para visualizar el diseño.
